# rig v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first read-only `rig` CLI that discovers Claude Code configuration roots, inventories components, and validates known config files through `claude-config-schema`.

**Architecture:** `rig` is a Go single binary with a small internal package graph. v0.1 intentionally avoids writes, snapshots, hooks, and the TUI; it proves config-root discovery, inventory normalization, schema-backed validation, and machine-readable reporting.

**Tech Stack:** Go 1.24+, Cobra for CLI, standard-library filesystem walking, `claude-config-schema` Go bindings when available, fixture-based tests, GitHub Actions.

---

## Source Documents

- Product spec: `docs/rig-spec.md`
- Future dependency: `claude-config-schema` Phase 1 Go bindings
- Runtime boundary reference: `docs/spec.md` in Ground Control

## v0.1 Scope

Ship read-only commands only:

```text
rig doctor
rig ls
rig lint
```

Do not implement:

- TUI
- snapshots
- stow/hoist
- hook install/uninstall
- archetype scoring
- export/import

Those features depend on a trusted inventory model and should start after v0.1 proves discovery and validation.

```go
package configroot

import (
	"os"
	"path/filepath"
)

type Root struct {
	Path   string `json:"path"`
	Source string `json:"source"`
}

func Resolve() (Root, error) {
	if explicit := os.Getenv("CLAUDE_CONFIG_DIR"); explicit != "" {
		abs, err := filepath.Abs(explicit)
		if err != nil {
			return Root{}, err
		}
		return Root{Path: abs, Source: "CLAUDE_CONFIG_DIR"}, nil
	}

	home, err := os.UserHomeDir()
	if err != nil {
		return Root{}, err
	}
	return Root{Path: filepath.Join(home, ".claude"), Source: "default"}, nil
}
```

- [ ] **Step 5: Add discovery**

Create `internal/inventory/discover.go`:

```go
package inventory

import (
	"os"
	"path/filepath"
	"strings"
)

func Discover(root string) ([]Component, error) {
	var components []Component

	settingsPath := filepath.Join(root, "settings.json")
	if fileExists(settingsPath) {
		components = append(components, component(ComponentSettings, "settings", settingsPath))
	}

	agentsDir := filepath.Join(root, "agents")
	if dirExists(agentsDir) {
		entries, err := os.ReadDir(agentsDir)
		if err != nil {
			return nil, err
		}
		for _, entry := range entries {
			if entry.IsDir() || filepath.Ext(entry.Name()) != ".md" {
				continue
			}
			path := filepath.Join(agentsDir, entry.Name())
			name := strings.TrimSuffix(entry.Name(), filepath.Ext(entry.Name()))
			components = append(components, component(ComponentAgent, name, path))
		}
	}

	skillsDir := filepath.Join(root, "skills")
	if dirExists(skillsDir) {
		entries, err := os.ReadDir(skillsDir)
		if err != nil {
			return nil, err
		}
		for _, entry := range entries {
			if !entry.IsDir() {
				continue
			}
			path := filepath.Join(skillsDir, entry.Name(), "SKILL.md")
			if fileExists(path) {
				components = append(components, component(ComponentSkill, entry.Name(), path))
			}
		}
	}

	return components, nil
}

func component(kind ComponentKind, name string, path string) Component {
	return Component{
		ID:     string(kind) + ":" + name,
		Kind:   kind,
		Scope:  ScopeGlobal,
		Name:   name,
		Path:   path,
		Active: true,
	}
}

func fileExists(path string) bool {
	info, err := os.Stat(path)
	return err == nil && !info.IsDir()
}

func dirExists(path string) bool {
	info, err := os.Stat(path)
	return err == nil && info.IsDir()
}
```

```go
package validate

import (
	"encoding/json"
	"os"

	"github.com/phatblat/rig/internal/inventory"
)

func Validate(component inventory.Component) []inventory.Problem {
	if component.Kind != inventory.ComponentSettings {
		return nil
	}

	data, err := os.ReadFile(component.Path)
	if err != nil {
		return []inventory.Problem{{
			Severity: "error",
			Code:     "read_failed",
			Message:  err.Error(),
			Path:     component.Path,
		}}
	}

	var raw map[string]any
	if err := json.Unmarshal(data, &raw); err != nil {
		return []inventory.Problem{{
			Severity: "error",
			Code:     "invalid_json",
			Message:  err.Error(),
			Path:     component.Path,
		}}
	}

	if permissions, ok := raw["permissions"]; ok {
		if _, ok := permissions.(map[string]any); !ok {
			return []inventory.Problem{{
				Severity: "error",
				Code:     "invalid_settings_permissions",
				Message:  "permissions must be an object",
				Path:     component.Path,
			}}
		}
	}

	return nil
}
```

This skeleton is deliberately narrow. Replace it with `claude-config-schema` validation once the Go package exists.

```go
					if _, err := fmt.Fprintf(cmd.OutOrStderr(), "%s: %s: %s\n", problem.Severity, problem.Path, problem.Message); err != nil {
						return err
					}
				}
			}
			if problemCount > 0 {
				return fmt.Errorf("%d validation problem(s)", problemCount)
			}
			_, err = fmt.Fprintln(cmd.OutOrStdout(), "ok")
			return err
		},
	}
}
```
